# COR DA LANDING PAGE

Temos como objetivo ter uma cor que atrae o cliente e ao mesmo tempo, deixe ele confortavel.  Querendo navegar na nossa pagina.

Com isso, pensamos em cores conforme abaxi.  Veja se faz sentido:
- laranja que representa a criatividade;
- azul que trás a confiança;
- tom bem fraco de cinza
